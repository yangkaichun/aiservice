Azure部屬方式

本系統採用微軟Azure雲端平台部署，確保高可用性、可擴展性與資安防護。透過Azure多元服務，我們能快速構建強大的智能客服解決方案，同時控制成本並提供彈性的擴展能力。

Azure Bot Service
提供對話機器人框架，整合多平台部署功能，確保系統穩定運行與智能對話體驗。
Azure Cognitive Services
提供語音識別、合成和自然語言處理等AI服務，強化多語言支援與高品質語音體驗。
Azure Cosmos DB
全球分布式資料庫，提供毫秒級低延遲與無限擴展能力，適合存儲大量醫療資訊。
Azure Security Center
提供全方位的安全監控與保護，符合醫療資料的嚴格安全標準與法規要求。
Azure Functions
無伺服器運算服務，按需執行程式碼，簡化開發流程並降低基礎設施維護成本。
Azure Monitor
全面的監控解決方案，提供即時系統性能分析、警報與報表功能，確保服務品質。
